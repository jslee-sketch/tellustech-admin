# 환영합니다

Tellustech 의 IT 렌탈·계측기 렌탈·정비 서비스를 이용해 주셔서 감사합니다.

**고객 포탈**은 다음 4가지를 한 곳에서 처리할 수 있는 온라인 창구입니다.

1. **AS 요청** — 본인 보유 장비 고장 신고
2. **소모품 요청** — 토너·잉크·부품 등 호환 품목 주문
3. **사용량 컨펌** — 월별 청구 사용량 확인 후 모바일 서명
4. **교정 성적서** — 검교정 PDF 다운로드

PC·태블릿·스마트폰 어느 화면에서나 동일하게 작동하며, 한국어 / 베트남어 / 영어 3개 언어를 지원합니다.

> 본 가이드는 **고객사 담당자**가 포탈을 사용할 때 참고합니다. 사내 직원용 ERP 매뉴얼(별책 A)과는 다른 별도 문서입니다.

# 1. 시작하기 — 접속과 로그인

## 1.1 포탈 접속

브라우저(Chrome / Safari / Edge)에서 Tellustech 담당자가 안내한 포탈 주소로 접속합니다.

```
https://portal.tellustech.co.kr
```

> 정확한 도메인은 담당자에게 확인하세요. 사내 직원용 도메인과 분리되어 있습니다.

## 1.2 로그인 정보

| 입력 항목 | 예시 | 비고 |
|---|---|---|
| **고객코드** | `CL-260101-001` | Tellustech 가 발급한 코드 (CL- 로 시작) |
| **비밀번호** | (담당자 발급) | 첫 로그인 후 즉시 변경 권장 |
| **언어** | 한국어 / Tiếng Việt / English | 화면 표기 언어 |

> **회사코드(TV/VR) 입력은 필요 없습니다.** 고객 포탈은 사내 직원 화면과 다른 입력 폼을 사용합니다.

## 1.3 로그인 화면 진입

로그인 페이지에서 화면 하단의 **🛒 고객 포탈 로그인 / Customer Portal** 버튼을 누르거나, 안내된 URL 에 `?portal=1` 파라미터가 붙어 있으면 자동으로 포탈 모드로 열립니다.

## 1.4 비밀번호 분실

비밀번호를 잊으신 경우 Tellustech 담당자에게 연락해 재설정을 요청해 주세요. 보안상 자가 재설정은 제공하지 않습니다.

# 2. 메인 화면 한눈에 보기

로그인 직후 표시되는 **메인 화면**은 4개 영역으로 구성됩니다.

## 2.1 헤더 — 회사명 / 고객코드

화면 상단에 귀사의 회사명과 고객코드(`CL-...`)가 표시됩니다.
**미수금 정지 상태**(상환 지연 등)인 경우 빨간색 `정지 / Bị khóa` 배지가 표시되며, AS·소모품 신규 요청이 일시적으로 비활성화됩니다. 해제는 결제 확인 후 자동 처리되거나 담당자에게 문의해 주세요.

## 2.2 빠른 요청 카드 — 4개

- 🛠 **AS 요청 / Yêu cầu BH**
- 📦 **소모품 요청 / Yêu cầu vật tư**
- ✍️ **사용량 컨펌 / Xác nhận sử dụng**
- 📄 **교정 성적서 / Chứng chỉ** (괄호 안 숫자는 다운로드 가능 건수)

각 항목을 누르면 해당 화면으로 이동합니다.

## 2.3 내 IT 계약 카드

활성/임박 IT 계약 최근 10건을 계약번호 순으로 표시합니다. 각 줄에는 **계약번호**, **상태**(`ACTIVE` / `DRAFT` 등), **장비 수**가 함께 보입니다.

## 2.4 내 요청 현황 표

직전에 등록한 AS·소모품 요청 최근 20건의 진행 상황입니다. 4단계로 흐릅니다.

| 단계 | 의미 |
|---|---|
| **요청 / Yêu cầu** | 접수 완료, 담당자 배정 대기 |
| **진행중 / Đang xử lý** | 담당자 배정·작업 진행 중 |
| **완료 / Hoàn tất** | 작업 완료, 고객 확인 대기 |
| **확인됨 / Đã xác nhận** | 고객이 확인 → 종결 |

상태가 `완료`인 줄의 **확인** 버튼을 누르면 종결 처리됩니다.

# 3. AS 요청

장비가 고장났거나 점검이 필요할 때 사용합니다.

## 3.1 장비 선택

화면 상단의 **장비 SN** 드롭다운에는 귀사가 현재 사용 중인 IT 계약·TM 렌탈 장비만 자동으로 표시됩니다. 형식은 다음과 같습니다.

```
SN-12345 · ITM-260101-001 LaserPrinter X7500 (TLS-260101-001)
```

순서대로 **시리얼 번호 · 품목코드 품목명 (계약번호)** 입니다. 본인 장비가 보이지 않으면 담당자에게 문의해 주세요.

## 3.2 증상 입력 — 한 언어로만 작성하면 됩니다

**원문 언어**(VI / KO / EN)를 먼저 선택한 뒤, 해당 언어 칸에 증상을 자유롭게 적어 주세요.

> 저장 시 시스템이 자동으로 나머지 2개 언어로 번역해 보관합니다. 한국어로 적은 내용도 베트남어 담당자가 읽을 수 있습니다. 세 칸 모두 채울 필요는 없습니다.

## 3.3 접수와 알림

**AS 접수 / Tiếp nhận BH** 버튼을 누르면 접수번호(예: `26/04/27-01`)가 발급되고, 등록된 이메일 또는 카카오톡으로 진행 단계마다 알림이 전송됩니다.

## 3.4 미수금 정지 시

회사 헤더에 정지 배지가 떠 있는 경우 AS 요청 메뉴는 회색으로 비활성화됩니다. 결제 확인 후 자동 해제됩니다.

# 4. 소모품 요청

토너·잉크·부품 등 정기 소모품을 요청합니다.

## 4.1 호환 품목 자동 필터

귀사의 IT 계약·TM 렌탈 장비에 **호환되는 품목만** 목록에 표시됩니다. 호환 매핑은 Tellustech 가 사전에 등록해 둡니다. 원하는 품목이 보이지 않으면 담당자에게 매핑 추가를 요청해 주세요.

## 4.2 입력 항목

| 항목 | 설명 |
|---|---|
| **품목 / Vật tư** | 드롭다운에서 선택. 표시는 `품목코드 · 품목명 (단위) [카테고리]` |
| **수량 / SL** | 1 이상 정수 |
| **메모 / Ghi chú** | 색상·모델·긴급 여부 등 자유 메모 |

`+ 품목 추가 / Thêm vật tư` 버튼으로 한 요청에 여러 품목을 함께 담을 수 있습니다.

## 4.3 접수

**요청 전송 / Gửi yêu cầu** 버튼을 누르면 단일 접수번호로 묶여 발급되며, 메인 화면 「내 요청 현황」에 즉시 나타납니다.

# 5. 사용량 컨펌

TM(계측기) 렌탈 장비의 월별 카운터를 확인하고 청구 금액을 컨펌합니다.

## 5.1 표 항목

| 컬럼 | 의미 |
|---|---|
| **월 / Tháng** | 청구 월 (예: `2026-04`) |
| **S/N** | 장비 시리얼 |
| **흑백 / B&W** | 흑백 카운터 누적값 |
| **컬러 / Color** | 컬러 카운터 누적값 |
| **금액 / Phí** | 시스템이 계산한 당월 청구액 |

## 5.2 모바일 손가락 서명 절차

1. 청구 줄 우측의 **✍️ 서명 / Ký tên** 버튼을 누릅니다.
2. 모달이 열리면 마우스(또는 스마트폰 손가락)로 서명 영역에 직접 그립니다.
3. **확인** 버튼으로 서명을 저장합니다. (✓ 표시가 나타납니다)
4. 옆의 **확인 / Xác nhận** 버튼을 누르면 컨펌이 완료됩니다.

> 한 번 컨펌하면 수정할 수 없습니다. 카운터 값에 이의가 있을 경우 컨펌 전에 Tellustech 담당자에게 연락해 주세요.

## 5.3 정지 상태와 무관

사용량 컨펌은 미수금 정지 상태와 관계없이 항상 사용할 수 있습니다.

# 6. 교정 성적서 다운로드

교정(검교정) 작업이 완료된 장비의 성적서 PDF 를 검색·다운로드합니다.

## 6.1 검색 조건 5가지 — 비워 두면 전체 조회

| 조건 | 입력 예 |
|---|---|
| **S/N** | `SN-12345` (부분 일치) |
| **성적서번호 / Số CC** | `CC-260101-001` |
| **품목 / Vật phẩm** | `ITM-...` 또는 품목명 일부 |
| **시작일 / Từ** | `2026-01-01` |
| **종료일 / Đến** | `2026-04-30` |

## 6.2 결과 표

| 컬럼 | 의미 |
|---|---|
| **발행일** | 성적서가 발급된 날짜 |
| **성적서번호** | 검교정 기관 발급 번호 |
| **S/N** | 장비 시리얼 |
| **품목** | 장비 정보 |
| **다음 기한** | 다음 정기 검교정 권장 일자 |
| **다운로드** | 📄 PDF 버튼 |

📄 PDF 버튼을 누르면 새 창에서 성적서가 열립니다. 브라우저의 인쇄·다운로드 메뉴로 저장하세요.

# 7. 모바일 PWA 설치

포탈은 PWA(Progressive Web App) 로 동작하므로 스마트폰 홈 화면에 앱처럼 설치할 수 있습니다.

## 7.1 iOS Safari

1. Safari 로 포탈 주소에 접속해 로그인까지 완료합니다.
2. 하단 가운데 **공유** 버튼(□↑) 을 누릅니다.
3. **「홈 화면에 추가」** 를 선택합니다.
4. 이름을 확인하고 **추가** 를 누르면 홈 화면에 아이콘이 생성됩니다.

## 7.2 Android Chrome

1. Chrome 으로 포탈 주소에 접속해 로그인까지 완료합니다.
2. 우상단 **⋮ 메뉴 → 「홈 화면에 추가」** 를 선택합니다.
3. **설치** 를 누르면 앱 서랍·홈에 아이콘이 생성됩니다.

## 7.3 앱처럼 사용하기

- 설치 후에는 브라우저 주소창 없이 전체 화면으로 실행됩니다.
- 일부 화면(메인·내 요청 현황 등)은 오프라인에서도 마지막 본 데이터를 캐시로 표시합니다. 새 요청·실시간 상태 갱신은 인터넷 연결이 필요합니다.

# 8. FAQ

**Q. 비밀번호를 잊었습니다.**
A. Tellustech 담당자에게 연락해 재설정을 요청해 주세요. 자가 재설정은 제공하지 않습니다.

**Q. AS 요청 진행 상황은 어디서 봅니까?**
A. 메인 화면 「내 요청 현황」 표에서 4단계(요청 → 진행중 → 완료 → 확인됨)로 추적됩니다. 알림은 등록된 이메일·카카오로 함께 전송됩니다.

**Q. 소모품 목록에 원하는 품목이 보이지 않습니다.**
A. 호환 매핑이 등록되지 않았을 가능성이 큽니다. 담당자에게 해당 품목을 본인 장비에 매핑해 달라고 요청해 주세요.

**Q. 사용량 컨펌에서 카운터 값이 잘못된 것 같습니다.**
A. **컨펌 버튼을 누르기 전에** 담당자에게 연락해 주세요. 컨펌이 완료되면 수정할 수 없습니다.

**Q. 회사 헤더에 빨간 정지 배지가 떠 있습니다.**
A. 미수금 결제가 지연되어 신규 요청 메뉴가 일시 비활성화된 상태입니다. 결제 확인 후 자동 해제됩니다. 사용량 컨펌·교정 성적서 다운로드는 정지와 무관하게 사용 가능합니다.

**Q. 알림은 어디로 옵니까?**
A. 등록된 이메일 / 카카오톡으로 전송됩니다. 수신 채널 변경은 담당자에게 문의해 주세요.

**Q. 한국어와 베트남어를 동시에 보고 싶습니다.**
A. 입력 시에는 원문 언어 한 개만 선택해 작성하면 시스템이 자동으로 나머지 언어를 번역해 보관합니다. 표시 언어는 화면 우상단 국기 아이콘으로 전환할 수 있습니다.
